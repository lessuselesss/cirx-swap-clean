Debug Test Page

What you should see on http://localhost:3000:

  • A dark background with a gradient overlay
  • Circular logo at the top
  • Tab selector: "Buy Liquid" and "Buy OTC with Discount"
  • Swap form with input fields
  • Connect Wallet button

Open Browser Developer Console (F12) and check:

  1. Console tab for JavaScript errors
  2. Network tab for failed resource loads (404 errors)
  3. Elements/Inspector tab to see if content exists but is hidden
